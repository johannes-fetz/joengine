Introduction :

Kronos est un émulateur Sega Saturn/ST-V en open source écrit en C/C++. Il est basé sur les sources de Yabause/UoYabause/Yaba Sanshiro, tle but est d'améliorer l'émulation et l’expérience Sega Saturn/ST-V.

Avant de lancer Kronos :

Pour avoir la meilleure expérience, suivez ces étapes :

  • Effacer votre bkram.bin --> par défaut, le chemin est C:\Users\yourname\Appdata\local\kronos\
  • Effacer votre ancien fichier kronos.ini ou utiliser le kronos.ini fournit dans l'archive.
  • Les anciennes sauvegardes/sauvegardes d'états ne devraient pas fonctionner avec Kronos 2.1.3.

OpenGL / Compute Shader video core :

Kronos a introduit un nouveau noyau vidéo : le noyau Compute Shader. Voici les différences avec le noyau OpenGL :

  • Noyau OpenGL : les cartes graphiques supportés < OpenGl 4.3.
  • Noyay Compute Shader : les cartes graphiques supportés >= OpenGl 4.3.
Certains problèmes graphiques ne pourront pas ^étre corrigés avec le noyau OpenGL (ex : les trous dans le route pour le jeu Sega Rally holes).

Fonctionnalités spécifiques à Kronos :

  • Support du Compute Shader.
  • Support de Windows 7/10.
  • Support des jeux ST-V.
  • Support de Kronos dans Retroarch (préliminaire).
  • Support des archives Redump au format Zip (si vous possédez un jeu Saturn provenant de chez Redump, il est possible de lancer le jeu sans décompresser l'archive).
  • Nouvelles options vidéo.
  • Support des filtres vidéos (HQ4X, Bicubic, BRZ2, etc...).
  • Nouvel interpréteur kronos (utilisation réduite du CPU).
  • Support des images CHD.
  • Plus de jeux ont été corrigés.
  • Suppression d'options inutiles et de vieux code.

Comment lancer un jeu STV ?

Pour lancer une jeu ST-V, veuillez suivre les étapes suivantes (port Windows) :

  • Créer un répertoire avec vos roms ST-V (ne surtout pas les décompresser). Ajout le Bios ST-V dans le même répertoire.
  • Lancer Kronos.exe
  • Cliquez sur "Fichier" --> "Paramètres" --> puis sur l'onglet "Cartouche/Mémoire".
  • Dans "Cartouche", sélectionnez "Rom Jeu ST-V".
  • Cliquez sur "..." pour ajouter le chemin du répertoire ou se trouve les roms ST-V. Patientez quelques secondes. Les jeux devraient être affichés sur la ligne au dessus.
  • En-dessous, dans Mémoire, donnez le chemin d'un fichier .bin utilisé uniquement pour le book keeping des jeux ST-V (un jeu ST-V peut avoir un comportement anormal en présence de la sauvegarde de son port Saturn).
  • Dans Contrôleur, choisir une configuration libre pour le Port 1 et sélectionner "Contrôles ST-V". Cliquez sur l'icône de clé à molette à côté et réglez chaque bouton du joueur 1.
  • Répéter l'opération pour le Port 2 pour jouer à 2.
  • Cliquez sur "Ok" pour fermer la fenêtre des "Paramètres".
  • Pressez l'icône "Lancer" dans la barre d'outils ou dans le menu "Emulation".
